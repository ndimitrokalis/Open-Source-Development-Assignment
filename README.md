# LibreHelp

LibreHelp is an open-source customer support system designed to help organizations manage customer inquiries through a structured ticketing system and knowledge base.

The goal of LibreHelp is to provide a lightweight, easy-to-deploy helpdesk platform that allows businesses and teams to organize support requests efficiently while maintaining a clean and simple interface.

This project is being developed as part of an Open-Source Software Development assignment and demonstrates the process of planning, developing, and documenting an open-source software project.

---

## ✨ Features

LibreHelp aims to provide the following core features:

* Customer support ticket management
* Ticket status tracking (open, in progress, closed)
* Agent and customer user roles
* Knowledge base for frequently asked questions
* Web interface for managing tickets and articles
* Simple and modular architecture

Future improvements may include additional features such as improved search functionality, enhanced user roles, and interface improvements.

---

## 🛠 Technology Stack

LibreHelp is built using the following technologies:

**Backend**

* Python
* Flask
* SQLAlchemy
* Flask-Login

**Frontend**

* Bootstrap
* Jinja Templates

**Database**

* SQLite (development)
* MySQL (optional for production)

**Project Infrastructure**

* GitHub for version control
* Jira for project management
* GitHub Wiki for documentation
