# Code of Conduct

## 🤝 Our Commitment

The **LibreHelp** project is committed to providing a respectful and collaborative environment for all contributors.

We are committed to creating a community that is welcoming regardless of:

* experience level
* gender identity or expression
* nationality
* personal background
* education level

All contributors are expected to interact in a professional and constructive manner while participating in the project.

---

## 📋 Expected Behavior

Participants in this project are expected to:

* Be respectful and professional in communication
* Accept constructive feedback
* Focus on improving the project
* Collaborate in a positive and supportive way
* Contribute in a manner that benefits the community

Maintainers and contributors share responsibility for maintaining a productive and welcoming environment.

---

## 🛠 Enforcement Responsibilities

Project maintainers are responsible for enforcing the standards outlined in this Code of Conduct.

Maintainers may take appropriate action when behavior is found to violate these standards in order to maintain a respectful project environment.

---

## 📢 Reporting Issues

If a contributor experiences behavior that violates this Code of Conduct, they should report it to the project maintainers.

All reports will be reviewed and handled appropriately.

---

## ⚖ Consequences

Violations of this Code of Conduct will result in the following actions:

First offense:
A warning will be issued to the contributor.

Second offense:
The contributor will be removed from the project and their participation will be revoked.

---

## 📄 Acknowledgment

By participating in this project, contributors agree to follow the guidelines outlined in this Code of Conduct.
